## ğŸ”§ Customization

### Adding Products
1. Open the desired HTML file (collections.html, sarees.html, etc.)
2. Copy an existing product card structure
3. Update the image URL, title, description, and price
4. Ensure each product has a unique image URL

### Changing Colors
1. Open `styles.css`
2. Modify the CSS variables in the `:root` selector
3. Colors will update across the entire site

### Modifying Navigation
1. Update the `<nav>` section in each HTML file
2. Add or remove menu items as needed
3. Update active class for current page
